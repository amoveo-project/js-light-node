## Installation

```
yarn add amoveo-js-light-node
```

or

```
npm install --save amoveo-js-light-node
```

## Usage

```
import VeoNode from 'amoveo-js-light-node'

const veoNodeUrl = 'http://amoveo.exan.tech:8080';
const veo = new VeoNode(veoNodeUrl);
```
